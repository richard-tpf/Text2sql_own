## 二、指标定义

| 指标名称 | 字段性质 | 统计方式 | 取值来源 | 取值规则 | 展示格式 | 备注 |
|---------|---------|---------|---------|---------|---------|------|
| 日期 | 维度 | — | 来源表：`order_info`；字段：`create_date` | 取订单创建日期，按自然日截断 | YYYY-MM-DD | |
| 申请数 | 原子指标 | 计数（去重） | 来源表：`credit_apply`；字段：`user_id` | 当日提交授信申请的去重用户数，按 `user_id` 去重 | 整数 | |
| 授信通过数 | 原子指标 | 计数（去重） | 来源表：`credit_apply`；字段：`user_id`, `status` | 当日授信审批结果为"通过"的去重用户数，条件：`status = 'approved'` | 整数 | |
| 授信通过率 | 派生指标 | 比率 | 来源表：`credit_apply`（同上两个指标） | 授信通过数 / 申请数 × 100% | 百分比（保留2位小数） | 分母为0时展示"—" |
| （补充指标） | | | | | | |

> **字段性质说明**：维度、原子指标、派生指标、复合指标等。
>
> **统计方式说明**：计数、计数（去重）、求和、平均值、最大值、最小值、比率等。
>
> **取值来源填写规范**：
> - 列出取值来源表名及相关字段，格式为 `来源表：表名；字段：字段名`
> - 若涉及多表关联，需注明关联关系，例如：
>   `来源表：table_a LEFT JOIN table_b ON table_a.id = table_b.a_id；字段：table_a.amount`
