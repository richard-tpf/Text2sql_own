# demo02 架构概览

## 项目简介

基于 Vanna Agents 框架搭建的数仓开发助手。使用 DeepSeek-R1 大语言模型（通过 SiliconFlow 的 OpenAI 兼容接口调用），结合双 Milvus 向量知识库（表结构知识库 + 业务文档知识库），实现 DWD → DWS → ADS 分层架构下的数仓加工 SQL 开发。

与 demo01（通用知识库助手）相比，demo02 做了以下演进：
- 单知识库 → 双知识库（表结构 + 业务文档），各自独立的 Milvus 集合和数据模型
- 通用问答 → 数仓开发专用，内置分层架构规范、Hive SQL 编写标准、开发流程和检查清单
- 新增 Web 管理页面：表结构上传页（支持手动输入和 MD 文件批量导入）、业务文档上传页
- 新增模板下载功能：需求文档模板、开发文档模板、表结构导入模板
- 系统提示从通用知识库助手升级为数仓开发专用提示（DWSystemPromptBuilder）

## 技术栈

| 组件 | 技术选型 | 说明 |
|------|---------|------|
| 大语言模型 | DeepSeek-R1 | 通过 SiliconFlow API 调用，OpenAI 兼容接口 |
| 向量数据库 | Milvus (172.16.11.57:19530) | 双集合：dw_schema_kb + dw_business_kb |
| 嵌入模型 | BAAI/bge-m3 | 将文本转换为 1024 维向量 |
| Web 服务 | FastAPI + Uvicorn | 提供 HTTP API、前端页面和模板下载 |
| 基础框架 | Vanna Agents | 提供 Agent、工具注册、中间件等核心能力 |

## 注册的工具

| 工具 | 功能 | 可用用户组 |
|------|------|-----------|
| SaveKnowledgeTool | 保存表结构或业务文档到对应知识库 | admin, user |
| SearchKnowledgeTool | 搜索知识库（支持跨库、定向搜索、层级过滤） | admin, user |

## Web 页面

| 页面 | 路径 | 说明 |
|------|------|------|
| 对话主页 | `/` | 包含登录、对话、双知识库入口和模板下载 |
| 表结构上传 | `/knowledge/schema` | 手动输入 DDL 或上传 MD 文件批量导入 |
| 业务文档上传 | `/knowledge/business` | 上传业务知识、需求文档、开发规范 |

## API 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/knowledge/schema/upload` | 上传单条表结构 |
| POST | `/api/knowledge/schema/upload_batch` | 批量上传表结构 |
| POST | `/api/knowledge/business/upload` | 上传单条业务文档 |
| POST | `/api/knowledge/business/upload_batch` | 批量上传业务文档 |
| GET | `/api/templates/{template_name}` | 下载模板文件（schema/requirement/development） |
