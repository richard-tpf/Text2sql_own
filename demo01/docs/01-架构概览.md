# demo01 架构概览

## 项目简介

基于 Vanna Agents 框架搭建的知识库驱动智能助手。使用 DeepSeek-R1 大语言模型（通过 SiliconFlow 的 OpenAI 兼容接口调用），结合 Milvus 向量数据库实现知识库功能，能够存储建表语句和业务知识，并在用户提问时搜索知识库后生成回答。

与原版 xsrtest 相比，去除了历史记忆搜索和 SQL 执行功能，专注于知识库管理和问答。

## 技术栈

| 组件 | 技术选型 | 说明 |
|------|---------|------|
| 大语言模型 | DeepSeek-R1 | 通过 SiliconFlow API 调用，OpenAI 兼容接口 |
| 向量数据库 | Milvus (172.16.11.57:19530) | 存储知识库向量数据 |
| 嵌入模型 | BAAI/bge-m3 | 将文本转换为 1024 维向量 |
| Web 服务 | FastAPI + Uvicorn | 提供 HTTP API 和前端页面 |
| 基础框架 | Vanna Agents v2.0.2 | 提供 Agent、工具注册、中间件等核心能力 |

## 文件结构

```

├── KnowledgeSystemPromptBuilder.py # 知识库系统提示构建器
```
