## 项目简介

基于 Vanna Agents 框架搭建的自定义 text2sql 智能助手。它使用 DeepSeek-R1 大语言模型（通过 SiliconFlow 的 OpenAI 兼容接口调用），结合 Milvus 向量数据库实现记忆功能，能够将用户的自然语言问题转换为 SQL 查询并执行。

## 技术栈

| 组件 | 技术选型 | 说明 |
|------|---------|------|
| 大语言模型 | DeepSeek-R1 | 通过 SiliconFlow API 调用，OpenAI 兼容接口 |
| 向量数据库 | Milvus (172.16.11.57:19530) | 存储工具使用记忆和文本记忆 |
| 嵌入模型 | BAAI/bge-m3 | 将文本转换为 1024 维向量 |
| 业务数据库 | MySQL | 实际执行 SQL 查询的目标数据库 |
| Web 服务 | FastAPI + Uvicorn | 提供 HTTP API 和前端页面 |
| 基础框架 | Vanna Agents v2.0.2 | 提供 Agent、工具注册、中间件等核心能力 |
